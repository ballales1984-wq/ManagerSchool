===========================================
  ISTRUZIONI PER CARICARE SU GITHUB
===========================================

FASE 1: CREA IL REPOSITORY SU GITHUB
-------------------------------------
1. Vai su https://github.com/new
2. Nome repository: ManagerSchool
3. Descrizione: Sistema completo di gestione scolastica digitale
4. Scegli PRIVATO o PUBBLICO
5. NON inizializzare con README, .gitignore o licenza
6. Clicca "Create repository"

FASE 2: COLLEGA IL TUO PROGETTO
--------------------------------
Dopo aver creato il repository, GitHub ti mostrera' i comandi.
Oppure usa questi comandi:

git remote add origin https://github.com/TUO_USERNAME/ManagerSchool.git
git branch -M main
git push -u origin main

Oppure se usi SSH:
git remote add origin git@github.com:TUO_USERNAME/ManagerSchool.git

FASE 3: PUSH INIZIALE
----------------------
git push -u origin main

Poi per aggiornamenti futuri:
git add -A
git commit -m "Descrizione modifica"
git push
